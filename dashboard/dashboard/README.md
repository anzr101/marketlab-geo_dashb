# MarketLab Dashboard

Interactive dashboard for the MarketLab research project.

## Features

- 🏠 Home: Project overview
- 🌍 Event Classifier: Live classification demo
- 📊 Research Findings: Prediction-profit gap
- 🔬 Model Performance: 1,386 models analyzed
- 📄 About: Research overview

## Run Locally

```bash
streamlit run app.py
```
